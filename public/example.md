# Presentor

A markdown-powered presentation tool

---

# Why Presentor?

- Write slides in any text editor
- No accounts, no cloud, no bloat
- Just markdown and a browser

---

# How It Works

Write your slides in a .md file

Separate them with three dashes

Navigate with arrow keys

---

# Markdown Features

- Headings become slide titles
- Bullet points just work
- Plain text becomes paragraphs
- Images via standard markdown syntax

---

# Keyboard Shortcuts

- Arrow Right / Space — next slide
- Arrow Left — previous slide
- F — toggle fullscreen
- Escape — exit presentation

---

# Get Started

1. Write your slides in a .md file
2. Open Presentor
3. Load your file
4. Present!
